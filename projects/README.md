# Intern Project - User Authentication System

## Overview
This project is a web-based User Authentication System developed as part of a web development internship. It provides essential user account management features including registration, login, and profile management through a simple and user-friendly interface.

## Features
- User Registration
- User Login Authentication
- User Profile Management
- Responsive User Interface
- Frontend Validation using JavaScript
- Backend Processing using PHP
- Organized Project Structure

## Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- PHP

## Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/intern-project.git
```

2. Open the project folder

```bash
cd intern-project
```

3. Run the project using a local server such as:
- XAMPP
- WAMP
- Laragon

4. Place the project folder inside the server's root directory and start Apache.

## Usage

1. Open the application in a web browser.
2. Register a new user account.
3. Login using registered credentials.
4. Access and manage the user profile.
